# Plugin Hub Research

## How to Search for Prior Art

1. **Plugin Hub Search**: Browse https://runelite.net/plugin-hub for existing plugins by keyword.
2. **GitHub Search**: Search `org:runelite` or the `runelite/plugin-hub` repo for plugin names and manifests.
3. **Core Plugins**: Check RuneLite core plugins at https://github.com/runelite/runelite/tree/master/runelite-client/src/main/java/net/runelite/client/plugins for built-in implementations.

## When to Search

- Before creating a new plugin — check if an existing one covers the same functionality.
- Before adding a major feature — look for prior art in existing Plugin Hub plugins.
- When touching an unfamiliar API — find examples of how other plugins use it.

## Evaluating Overlap

- If a Plugin Hub plugin already does what you need, consider contributing to it instead of creating a duplicate.
- If your plugin extends or complements an existing one, note the relationship.
